Cricket Team Selection

The Cricket Team Selection Game is an exciting and interactive web application designed for cricket enthusiasts to build their dream team. Users begin with a virtual budget of $33, which they must manage carefully to create a balanced team. Players can select from four categories—Batsmen, Wicketkeepers, All-Rounders, and Bowlers—while adhering to specific constraints for each category. The game introduces dynamic elements such as real-time player stats and an intuitive drag-and-drop interface to finalize team arrangements. The application is fully responsive, offering a seamless experience across desktops and mobile devices. Once the team is formed, users can assign roles like Captain and Vice Captain to complete their dream lineup.

Built using HTML, CSS, and JavaScript, this project showcases modern web development practices, including dynamic interactions, responsive design, and an engaging user experience. The thoughtful integration of functionality and design makes this project a must-try for cricket lovers.

Weather App

The Weather App is a sleek and functional desktop application designed using PyQt5 and powered by the OpenWeatherMap API. It provides users with real-time weather updates for any city worldwide. With its intuitive graphical interface, users can enter a city name and instantly view the temperature, weather description, and even an emoji to represent the current weather conditions. The application ensures smooth user interaction with a well-structured layout, offering a seamless experience for weather enthusiasts and everyday users alike.

Developed as a personal project, the app includes essential features such as error handling for invalid city names and connection issues, ensuring reliability in all scenarios. With its clean and responsive design, the Weather App highlights practical use cases for Python programming and graphical interface development.

To-Do List Application

The To-Do List Application is a user-friendly and interactive task management tool built with React. This project enables users to efficiently manage their daily tasks by adding, deleting, and organizing them with ease. It also features drag-and-drop reordering for intuitive task prioritization, making it suitable for users of all experience levels.

The application supports priority-based sorting, allowing tasks to be arranged by importance, and its responsive design ensures seamless performance across devices. Designed with simplicity and functionality in mind, the To-Do List Application demonstrates the power of React and JavaScript in creating modern web applications with a polished interface.
